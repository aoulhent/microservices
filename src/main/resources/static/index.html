🗓️ API Gestion Rendez-vous

Bienvenue sur l'API REST pour la gestion de rendez-vous

Version 1.0.0

✅ État du serveur : L'API est opérationnelle et prête à être utilisée.

Endpoints disponibles

GET /api/appointments

Récupère tous les rendez-vous

POST /api/appointments

Crée un nouveau rendez-vous

GET /api/appointments/{id}

Récupère un rendez-vous par ID

PUT /api/appointments/{id}

Met à jour un rendez-vous

DELETE /api/appointments/{id}

Supprime un rendez-vous

GET /api/appointments/client/{email}

Récupère les rendez-vous par email du client

GET /api/appointments/health

Vérification de santé de l'API
